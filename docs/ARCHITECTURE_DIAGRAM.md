# System Architecture & Workflow Diagram

## Simple Workflow Diagram

```
┌─────────────────────────────────────────────────────────────┐
│                    PR Event Triggered                        │
│              (GitHub Actions / Bitbucket)                    │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                  Bot Initialization                          │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐        │
│  │ Load SRS     │  │ Detect JIRA │  │ Fetch JIRA  │        │
│  │ Documents    │  │ Ticket Key   │  │ Details      │        │
│  └──────────────┘  └──────────────┘  └──────────────┘        │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                    Code Analysis                             │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐        │
│  │ Get PR Diff  │  │ Build AI     │  │ AI Review    │        │
│  │ (Changed     │→ │ Prompt       │→ │ (OpenAI GPT) │        │
│  │  Files Only) │  │ (SRS+JIRA+  │  │              │        │
│  │              │  │  Code)       │  │              │        │
│  └──────────────┘  └──────────────┘  └──────────────┘        │
└───────────────────────────┬─────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                    Review Output                             │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐        │
│  │ Inline       │  │ Review       │  │ Final        │        │
│  │ Comments     │  │ Summary      │  │ Verdict      │        │
│  │ (Critical    │  │ (Structured)  │  │ (Approve/    │        │
│  │  Issues)     │  │              │  │  Changes)    │        │
│  └──────────────┘  └──────────────┘  └──────────────┘        │
└─────────────────────────────────────────────────────────────┘
```

